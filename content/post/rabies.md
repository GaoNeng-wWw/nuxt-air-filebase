# 狂犬病

狂犬病是一种有狂犬病病毒引起的人畜共患疾病, 可在恒温动物身上造成严重的脑炎. 对于未接种疫苗的患者, 神经症状出现时几乎100%死亡.

## 狂犬病毒

狂犬病毒属于弹状病毒科, 狂犬病毒属. 核衣壳呈螺旋对称, 内含有单链DNA. 狂犬病毒主要通过受感染动物的唾液传播, 少数由抓伤传播, 少有因器官移植而导致的狂犬病. 狂犬病毒**不会**突破完整的皮肤.

### 基因组

狂犬病毒是单股负链RNA. 狂犬病毒包含五种蛋白, 糖蛋白(G), 核蛋白(N), 双聚酶(L), 磷蛋白(NS), 基质(M).

| 蛋白名称 | 主要功能描述                                                                                                 |
|----------|-----------------------------------------------------------------------------------------------------------|
| 核蛋白   | 可溶性N-P复合物专门包裹基因组RNA，蛋白质N像珍珠项链一样保护基因组。包裹的基因组RNA被称为核衣壳（NC），并作为病毒转录和复制的模板. |
| 糖蛋白   | 将病毒附着在宿主细胞受体上，利用不同的宿主蛋白诱导病毒颗粒的内吞作用, 决定病毒的致病性和组织嗜性.                            |
| 基质     | 在病毒组装、形态发生、包膜病毒粒子出芽以及调节RNA合成并深刻影响复制和转录产物的平衡[[@mbs:/content/journal/jgv/10.1099/vir.0.19128-0]].                            |
| 双聚酶   | 催化病毒mRNA的转录、封盖和多聚腺苷酸化                                                        |
| 磷蛋白   | 在病毒转录和复制中起作用的非催化聚合酶辅因子和调节蛋白。将RNA聚合酶L稳定在N-RNA模板上，并结合可溶性蛋白N，防止其包裹非基因组RNA. |

### 生命周期

狂犬病病毒具有高度嗜神经性, 病毒自破损伤口或粘膜侵入身体后会先在伤口周围处繁殖. 狂犬病毒包面的糖蛋白与特定受体(如乙酰胆碱受体)结合后, 通过内吞作用进入局部肌肉或神经末梢细胞, 在细胞内释放RNA, 在局部可能停留 3 日或更久.

其后, 病毒沿轴突以逆轴浆运动的方向向中枢神经系统移行, 上行到背根神经节后, 在其内大量增殖, 然后侵入脊髓和其他中枢神经系统. 由于迷走、舌咽及舌下神经核受损、致吞咽肌及呼吸肌痉挛出现恐水. 吞咽及呼吸困难等症状. 迷走神经节、交感神经节和心脏神经节受损时, 可引起心血管功能紊乱或者猝死.

### 免疫

G蛋白是狂犬病病毒中最主要的抗原,利用不同的宿主蛋白诱导病毒颗粒的内吞作用, 决定病毒的致病性和组织嗜性.可以有效刺激特异性辅助性T细胞和细胞毒性T细胞增生, 并诱导机体产生特异性抗体.
